# meu-primeiro-pr

Repositório criado para praticar o fluxo completo de contribuição no GitHub:
criar um branch, fazer uma alteração e abrir um Pull Request.

## Como contribuir

1. Crie um branch a partir de `main`
2. Faça suas alterações
3. Abra um Pull Request descrevendo o que mudou

## Tecnologias

- Git
- GitHub CLI (`gh`)
